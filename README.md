## BEM命名法

1.  `块(block)` 与 `元素(element)` 之间使用 `双下划线__`链接，例如`form__item`
2. `元素(element)`与`状态(modifier)`之间使用`双中划线--`链接，表示元素的不同状态，例如`el__button--success`
3. 使用`js`控制样式时，`css`命名用`is-`开头，例如`is-success`
